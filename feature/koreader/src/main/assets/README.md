# KOReader Runtime Assets

Place KOReader Lua runtime assets here:

```
assets/
└── koreader/
    ├── reader.lua          (main entry point)
    ├── frontend/           (UI + application code)
    ├── plugins/            (KOReader plugins)
    ├── fonts/              (bundled fonts)
    └── resources/          (images, CSS, etc.)
```

These assets are copied from the vendored `vendor/koreader` build output.
